# Oculomotor Dysfunction After TBI

## Introduction

Concussion management in sports has become increasingly sophisticated, with visual function playing a central role in assessment and rehabilitation. This comprehensive guide addresses the intersection of vision and concussion, specifically focusing on visual effects of concussion within concussion and vision.

## Understanding Sports-Related Concussion

### Definition
A concussion is a complex pathophysiological process affecting the brain, induced by biomechanical forces. Visual system involvement is common due to:
- Direct trauma to visual pathways
- Disruption of vestibular-ocular connections
- Cognitive processing changes
- Accommodative and vergence dysfunction

### Visual Symptom Prevalence

Research indicates that up to 90% of concussed patients experience visual symptoms:

| Visual Symptom | Prevalence | Duration Range |
|----------------|------------|----------------|
| Blurred vision | 45-65% | Days to weeks |
| Light sensitivity | 50-70% | Days to months |
| Reading difficulties | 40-60% | Days to weeks |
| Eye strain | 35-55% | Variable |
| Double vision | 15-30% | Usually transient |
| Visual motion sensitivity | 25-45% | Weeks to months |

## Visual Effects of Concussion

### Oculomotor Dysfunction

Concussion commonly affects eye movement control:

**Saccadic Dysfunction:**
- Increased latency
- Reduced accuracy
- Slowed velocity
- Impaired inhibition

**Pursuit Abnormalities:**
- Reduced gain
- Increased saccadic intrusion
- Asymmetric tracking

**Convergence Insufficiency:**
- Receded near point of convergence
- Reduced convergence amplitudes
- Symptoms with near work

### Accommodative Dysfunction

Post-concussion accommodative changes include:
- Reduced amplitude of accommodation
- Slowed accommodative facility
- Accommodative spasm (less common)
- Difficulty with focus transitions

### Vestibular-Ocular Dysfunction

The vestibular-ocular reflex (VOR) is frequently affected:
- Impaired VOR gain
- VOR asymmetry
- Visual-vestibular mismatch symptoms
- Motion sensitivity

### Visual Processing Changes

Higher-level visual processing may be impacted:
- Slowed visual processing speed
- Reduced visual attention
- Impaired visual memory
- Pattern recognition deficits

## Assessment Protocols

### Baseline Testing

Ideally, pre-season baseline testing establishes individual normative values:

**Core Visual Measures:**
- Static visual acuity
- Contrast sensitivity
- Near point of convergence
- Vergence amplitudes
- Accommodative amplitude and facility
- Saccadic function
- Vestibular-ocular function

### Sideline Assessment

Rapid visual screening following suspected concussion:

**SCAT5 Visual Components:**
1. Orientation questions
2. Immediate memory assessment
3. Concentration evaluation
4. Balance examination
5. Delayed recall

**King-Devick Test:**
- Rapid number naming
- Time-based performance
- Pre/post comparison
- Error counting

### Post-Concussion Vision Evaluation

Comprehensive assessment should evaluate:

1. **Visual Acuity**
   - Distance and near
   - Corrected and uncorrected
   - Dynamic acuity if indicated

2. **Binocular Vision**
   - Phoria measurement
   - Vergence amplitudes
   - Near point of convergence
   - Stereopsis

3. **Accommodation**
   - Amplitude assessment
   - Facility testing
   - Relative accommodation

4. **Oculomotor Function**
   - Saccadic assessment
   - Pursuit evaluation
   - Fixation stability

5. **Vestibular-Ocular Function**
   - VOR assessment
   - Head thrust test
   - Dynamic visual acuity

6. **Visual Processing**
   - Processing speed
   - Visual attention
   - Pattern recognition

## Rehabilitation Strategies

### Vision Therapy Approaches

**Convergence Training:**
- Brock string exercises
- Convergence cards
- Computerized vergence training
- Stereoscopic targets

**Accommodative Training:**
- Lens flipper exercises
- Hart chart activities
- Near-far focus shifts
- Monocular and binocular protocols

**Oculomotor Rehabilitation:**
- Saccadic training
- Pursuit exercises
- Fixation stability drills
- Reading tracking exercises

### Vestibular Rehabilitation Integration

Collaboration with vestibular specialists for:
- VOR training exercises
- Gaze stability training
- Balance integration
- Motion desensitization

### Light Sensitivity Management

**Environmental Modifications:**
- Gradual light exposure
- Sunglasses with FL-41 tint
- Screen brightness adjustment
- Lighting optimization

**Therapeutic Approaches:**
- Progressive light exposure therapy
- Colored overlay trials
- Blue-light filtering lenses

## Return-to-Play Considerations

### Visual Criteria for Return

Before returning to competition, athletes should demonstrate:
- Baseline or near-baseline visual acuity
- Normal or corrected convergence function
- Normal accommodative function
- Intact oculomotor control
- Resolution of visual symptoms
- Normal vestibular-ocular function
- Successful graded exertion without symptom exacerbation

### Graduated Return Protocol

**Stage 1: Symptom-limited activity**
- Daily activities that do not provoke symptoms
- Visual tasks limited to tolerance

**Stage 2: Light aerobic exercise**
- Walking, swimming, stationary cycling
- No resistance training
- Visual stimuli minimal

**Stage 3: Sport-specific exercise**
- Skating drills, running drills
- No head impact activities
- Increased visual demands

**Stage 4: Non-contact training drills**
- Harder training drills
- May start progressive resistance training
- Complex visual-motor integration

**Stage 5: Full-contact practice**
- Following medical clearance
- Full visual-motor demands
- Competition simulation

**Stage 6: Return to competition**

## Multi-Disciplinary Team Approach

### Key Team Members
- Sports medicine physician
- Neurologist (if indicated)
- Optometrist/Vision specialist
- Vestibular therapist
- Athletic trainer
- Neuropsychologist
- Team physician

### Communication Protocols
- Regular team meetings
- Shared documentation
- Clear referral pathways
- Unified return-to-play decisions

## Long-Term Monitoring

Athletes with history of concussion should have:
- Annual visual function assessment
- Baseline re-establishment each season
- Education on recognizing recurrent symptoms
- Access to vision specialist for new symptoms

## Clinical Pearls

1. Always obtain pre-season baseline when possible
2. Visual symptoms are common but often overlooked
3. Comprehensive vision evaluation is essential post-concussion
4. Multidisciplinary care optimizes outcomes
5. Return-to-play decisions should include visual criteria
6. Some athletes may require prolonged rehabilitation

## Case Example

**Presentation:** A 20-year-old college soccer player sustained a concussion during a game. Initial symptoms included headache, dizziness, and blurred vision.

**Assessment Findings (Week 1):**
- Visual acuity: Normal
- Near point of convergence: 12cm (baseline 4cm)
- Accommodative amplitude: Reduced 4D from baseline
- Saccadic latency: Increased by 50ms
- Light sensitivity: Moderate

**Rehabilitation Program:**
- Convergence exercises (Brock string, 3x daily)
- Accommodative training (flippers, Hart chart)
- Gradual light exposure
- Vestibular therapy integration

**Outcome (Week 6):**
- Full return to baseline measures
- No symptoms with exertion
- Cleared for competition

## Conclusion

Vision plays a critical role in concussion assessment and management. By implementing comprehensive visual protocols and collaborating across disciplines, sports vision specialists can significantly contribute to athlete safety and recovery.
