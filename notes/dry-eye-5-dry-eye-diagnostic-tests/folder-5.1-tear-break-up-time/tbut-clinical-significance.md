# TBUT Measurement Technique

## Introduction

Tear Break-Up Time (TBUT), also known as Tear Film Break-Up Time (TFBUT), is a fundamental diagnostic test for evaluating tear film stability. It measures the time interval between a complete blink and the appearance of the first dry spot in the pre-corneal tear film.

## Clinical Significance

TBUT is a key diagnostic indicator for:
- Tear film instability
- Dry eye disease (both aqueous deficient and evaporative)
- Mucin deficiency
- Lipid layer dysfunction

**Normal Values:**
- Normal: ≥10 seconds
- Borderline: 6-10 seconds
- Abnormal: <5-6 seconds

## Fluorescein TBUT Technique

### Materials Required
- Fluorescein strip
- Saline solution
- Slit lamp biomicroscope
- Cobalt blue filter
- Timer

### Step-by-Step Procedure

**Dye Application:**
1. Moisten fluorescein strip with saline
2. Patient looks up
3. Touch strip to inferior bulbar conjunctiva
4. Avoid touching the cornea
5. Patient blinks several times

**Measurement:**
1. Set slit lamp to low magnification
2. Install cobalt blue filter
3. Ask patient to hold eyes open
4. Start timer after last blink
5. Observe for first dry spot (dark area)
6. Stop timer and record

**Repeat Testing:**
- Perform 3 measurements
- Calculate the average
- Measure both eyes

## Interpretation

| TBUT Value | Interpretation |
|------------|----------------|
| ≥10 seconds | Normal |
| 6-10 seconds | Borderline |
| <5 seconds | Abnormal |
| <3 seconds | Severely reduced |

## Non-Invasive TBUT (NITBUT)

### Advantages
- No dye required
- More reproducible
- Better correlation with symptoms

### Methods
- Placido disc topography
- Keratometer-based
- Specialized instruments (Keratograph)

## Clinical Pearls for Exams

1. Normal TBUT is ≥10 seconds
2. Use cobalt blue filter
3. Perform before other tests
4. Average 3 measurements
5. Non-invasive TBUT is more reproducible
6. Reduced TBUT indicates tear film instability
