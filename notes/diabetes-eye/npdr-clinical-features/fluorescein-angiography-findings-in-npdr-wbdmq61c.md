
# Fluorescein Angiography Findings in NPDR

## Introduction

Recognizing the clinical features of Non-Proliferative Diabetic Retinopathy is fundamental to optometric practice. This comprehensive review details the characteristic findings, examination techniques, and clinical patterns that define Diabetes and the Eye.

## Primary Clinical Features

### Characteristic Findings

The hallmark features of Non-Proliferative Diabetic Retinopathy include:

**Anterior Segment Findings**:
- Conjunctival changes: Variable involvement
- Corneal manifestations: Multiple possible changes
- Iris abnormalities: May be present in some conditions
- Lens changes: Cataract associations common

**Posterior Segment Findings**:
- Retinal vascular changes: Often the primary manifestation
- Retinal tissue changes: Edema, exudates, hemorrhage
- Optic nerve involvement: Various patterns possible
- Vitreous changes: May occur with progression

### Symptom Patterns

Patient-reported symptoms vary by disease stage:

| Stage | Common Symptoms | Visual Impact |
|-------|----------------|---------------|
| Early | Often asymptomatic | Minimal |
| Moderate | Blurring, distortion | Moderate |
| Advanced | Significant vision loss | Severe |
| Complicated | Acute changes | Variable |

## Detailed Examination Findings

### Fundus Examination

Systematic examination reveals:

1. **Vascular assessment**:
   - Arteriolar changes
   - Venous abnormalities
   - Capillary alterations
   - Neovascularization assessment

2. **Retinal tissue evaluation**:
   - Edema presence and extent
   - Exudate characteristics and distribution
   - Hemorrhage types and patterns
   - Structural changes

3. **Optic nerve assessment**:
   - Disc appearance
   - Cup-to-disc ratio
   - Nerve fiber layer status
   - Peripapillary changes

### Ancillary Findings

Additional examination components:

- **Visual acuity**: May be affected variably
- **Visual fields**: Pattern defects may be present
- **Intraocular pressure**: May be elevated in some conditions
- **Pupillary responses**: Afferent defect possible

## Staging by Clinical Features

### Mild Disease Features

- Minimal visible changes
- Subtle vascular alterations
- No vision impact
- Stable examination

### Moderate Disease Features

- Clearly visible findings
- Multiple features present
- Some visual symptoms
- May be progressive

### Severe Disease Features

- Extensive changes
- Significant tissue involvement
- Visual function affected
- Risk of progression high

## Differential Diagnosis

Conditions to consider in differential:

| This Condition | Mimicking Condition | Distinguishing Features |
|----------------|---------------------|------------------------|
| Characteristic finding | Similar appearance | Specific features |
| Systemic correlation | May lack correlation | Laboratory testing |
| Typical distribution | Different pattern | Imaging findings |
| Expected progression | Different course | Serial examination |

## Documentation Guidelines

Proper documentation includes:

1. **Description**: Detailed findings description
2. **Extent**: Geographic distribution
3. **Severity**: Qualitative assessment
4. **Change**: Comparison to previous
5. **Photos**: Documented when possible
