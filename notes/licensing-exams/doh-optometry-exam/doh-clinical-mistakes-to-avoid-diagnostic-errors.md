# DOH Clinical Mistakes to Avoid - Diagnostic Errors

## Introduction

Understanding common pitfalls helps avoid mistakes in the DOH Abu Dhabi examination.

## Question-Answering Pitfalls

### Common Wrong Answer Traps

| Trap Type | Description |
|-----------|-------------|
| Partially Correct | True but doesn't fully answer |
| Opposite Answer | Inverted relationship |
| Common Misconception | Frequently believed error |
| Too Extreme | Absolute statements |
| Too Similar | Confusing similar terms |

## Diagnostic Errors

| Error | Prevention |
|-------|------------|
| Premature closure | Generate full differential |
| Anchoring | Re-evaluate with new data |
| Availability bias | Consider prevalence |
| Confirmation bias | Look for disconfirming signs |

## Time Management Errors

| Error | Solution |
|-------|----------|
| Spending too long | Flag and move on |
| Not reviewing | Reserve time for review |
| Rushing | Read carefully |
| Missing questions | Use review screen |
