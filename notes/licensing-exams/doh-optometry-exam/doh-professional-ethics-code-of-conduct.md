# DOH Professional Ethics - Code of Conduct

## Introduction

This guide covers local regulations for optometrists under DOH Abu Dhabi jurisdiction.

## Scope of Practice

### Permitted Procedures

| Category | Procedures |
|----------|------------|
| Examination | Comprehensive eye exams |
| Refraction | Subjective and objective |
| Contact Lenses | Fitting and follow-up |
| Vision Therapy | Non-surgical treatment |
| Diagnostic | Visual fields, tonometry |

### Prohibited Activities

- Surgery
- Injection administration
- Oral medication prescription
- Laser procedures

## Prescribing Rights

| Drug Class | Examples |
|------------|----------|
| Anti-infectives | Fluoroquinolones |
| Anti-inflammatories | Steroids (low potency) |
| Antihistamines | Olopatadine |
| Lubricants | Various |
| Mydriatics | Tropicamide |

## Referral Protocols

### Urgent Referrals

| Condition | Timeframe |
|-----------|-----------|
| Acute angle-closure glaucoma | Immediate |
| Retinal detachment | Same day |
| Orbital cellulitis | Immediate |
| Corneal ulcer | 24 hours |

## Professional Ethics

1. Patient welfare first
2. Honesty and integrity
3. Confidentiality
4. Professional boundaries
