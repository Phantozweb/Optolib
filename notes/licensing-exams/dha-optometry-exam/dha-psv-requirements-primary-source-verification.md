# DHA PSV Requirements - Primary Source Verification

## Overview

This comprehensive guide covers the registration process for the DHA Dubai optometry licensing examination. Understanding the registration requirements and procedures is essential for a successful application.

## Eligibility Requirements

### Educational Qualifications

To be eligible for the DHA Dubai optometry licensing examination, candidates must meet the following educational requirements:

| Requirement | Details |
|-------------|---------|
| Primary Degree | Bachelor's degree in Optometry (B.Optom) or equivalent |
| Degree Duration | Minimum 4 years of full-time study |
| Institution | Recognized/accredited optometry program |
| Clinical Hours | Minimum 1,000 supervised clinical hours |

### Documentation Requirements

**Primary Documents:**
1. Valid passport (minimum 6 months validity)
2. Recent passport-sized photographs (white background)
3. Original degree certificate
4. Academic transcripts (all years)
5. Internship/completion certificate

**Verification Documents:**
1. Good Standing Certificate from licensing authority
2. Experience certificates from employers
3. Professional reference letters (minimum 2)
4. Police clearance certificate
5. Medical fitness certificate

## Registration Steps

### Step 1: Online Registration

1. Visit the official DHA Dubai portal
2. Create a new account with valid email
3. Complete personal information section
4. Upload required documents
5. Pay application fee

### Step 2: Document Verification

**Primary Source Verification (PSV):**
- All educational credentials verified through DataFlow
- Processing time: 2-4 weeks
- Verification fee applicable

### Step 3: Application Review

| Stage | Duration | Activity |
|-------|----------|----------|
| Initial Review | 5-7 days | Document completeness check |
| PSV Processing | 14-28 days | Credential verification |
| Final Review | 3-5 days | Application approval |

### Step 4: Examination Scheduling

Upon approval:
1. Receive scheduling permit via email
2. Select preferred test date and location
3. Confirm appointment
4. Print admission letter

## Fee Structure

| Fee Type | Amount (USD) | Notes |
|----------|--------------|-------|
| Application Fee | $100-200 | Non-refundable |
| PSV Fee | $200-300 | Per credential |
| Examination Fee | $300-500 | Per attempt |
| License Fee | $200-400 | Upon passing |

## Common Application Issues

### Document Rejection Reasons

1. **Incomplete transcripts** - All pages required
2. **Expired passport** - Must be valid 6+ months
3. **Unclear copies** - High-resolution scans needed
4. **Missing attestation** - Proper attestation required
5. **Name mismatch** - Consistent spelling across all documents
