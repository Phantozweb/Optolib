# DOH Abu Dhabi Optometry Exam Question Types

## Overview

The Department of Health (DOH) Abu Dhabi optometry licensing examination employs a comprehensive assessment methodology designed to evaluate candidates' clinical knowledge, diagnostic reasoning, and professional competencies. Understanding the question types is essential for effective preparation.

## Primary Question Formats

### 1. Single Best Answer (SBA) Multiple Choice Questions

The predominant format in the DOH optometry exam, SBA questions present:
- A clinical scenario or direct question
- Five answer options (A-E)
- One clearly correct answer
- Distractors designed to test common misconceptions

**Example Structure:**
```
A 45-year-old patient presents with gradual onset of blurred vision and 
halos around lights. Intraocular pressure is 28 mmHg in the right eye 
and 18 mmHg in the left. The most likely diagnosis is:

A. Acute angle-closure glaucoma
B. Primary open-angle glaucoma
C. Ocular hypertension
D. Pigmentary glaucoma
E. Pseudoexfoliative glaucoma

Correct Answer: B
```

### 2. Multiple Response Questions (MRQs)

These questions require selection of multiple correct answers:
- Typically 5-7 options provided
- 2-4 correct answers required
- Partial credit may be awarded
- Tests comprehensive understanding of topics

**Example:**
```
Which of the following are contraindications for cycloplegic refraction 
in a pediatric patient? (Select all that apply)

A. Down syndrome
B. History of angle-closure glaucoma
C. Myopia greater than -6.00D
D. Anterior uveitis
E. Narrow anterior chamber angle

Correct Answers: B, E
```

### 3. Clinical Vignette Questions

Extended clinical scenarios testing:
- Diagnostic reasoning
- Clinical decision-making
- Management planning
- Referral criteria

These questions typically include:
- Detailed patient history
- Clinical findings
- Test results
- Multiple related questions

### 4. Image-Based Questions

Questions incorporating:
- Fundus photographs
- OCT scans
- Visual field plots
- Slit lamp photographs
- Corneal topography maps

**Common Image Types:**
- Diabetic retinopathy staging
- Glaucomatous optic nerve changes
- Macular pathology
- Anterior segment abnormalities

## Difficulty Levels

### Basic Level (30%)
- Direct knowledge recall
- Standard clinical procedures
- Common conditions

### Intermediate Level (50%)
- Application of knowledge
- Clinical reasoning
- Multi-step problem solving

### Advanced Level (20%)
- Complex case analysis
- Integration of multiple concepts
- Uncommon presentations

## Time Allocation

- **Total Questions:** 100-150 MCQs
- **Time Per Question:** Approximately 1.5-2 minutes
- **Total Duration:** 3-4 hours
- **Break Periods:** Optional 10-15 minute breaks

## Key Strategies for Different Question Types

### For SBA Questions:
1. Read the entire question carefully
2. Identify key clinical clues
3. Rule out obviously incorrect options
4. Select the BEST answer, not just a correct one
5. Avoid second-guessing unless clear error

### For MRQs:
1. Evaluate each option independently
2. Don't assume correlation between options
3. Mark clearly all correct answers
4. Review selections before submitting

### For Clinical Vignettes:
1. Take notes on key findings
2. Create a differential diagnosis list
3. Consider regional disease prevalence
4. Apply local practice guidelines

## Common Distractor Patterns

Understanding how distractors are created helps avoid common traps:

1. **Partially Correct Answers:** Statements that are true but don't fully answer the question
2. **Opposite Answers:** Correct statement with wrong modifier (e.g., "increases" instead of "decreases")
3. **Common Misconceptions:** Frequently believed but incorrect statements
4. **Similar-Sounding Terms:** Confusing similar terminology (e.g., myopia vs. myokymia)
5. **Regional Variations:** Practices that differ from UAE standards

## Computer-Based Testing Features

The Prometric testing platform includes:
- **Review Screen:** Flag questions for later review
- **Timer Display:** Remaining time visible
- **Navigation:** Move freely between questions
- **Highlighting:** Mark key information in questions
- **Strike-through:** Eliminate obviously wrong options

## Exam Day Recommendations

1. Arrive 30 minutes early for check-in
2. Bring valid identification (passport, Emirates ID)
3. Familiarize yourself with the testing interface
4. Use all available time - review flagged questions
5. Maintain steady pace throughout

## Post-Exam Score Reporting

- **Passing Score:** Typically 60-65% (varies by cycle)
- **Score Report:** Detailed breakdown by topic area
- **Retake Policy:** 3-month waiting period
- **Validity:** Scores valid for 2 years for licensing
